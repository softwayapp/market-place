# API Generator Skill

자동화된 RESTful API 엔드포인트 생성 스킬입니다.

## 빠른 시작

```bash
# 스킬 설치
/plugin install api-generator@internal-marketplace

# 사용 예제
"User 모델에 대한 REST API를 생성해줘"
```

## 주요 기능

- ✅ CRUD 엔드포인트 자동 생성
- ✅ 입력 검증 미들웨어
- ✅ Swagger/OpenAPI 문서 자동 생성
- ✅ 페이지네이션, 필터링, 정렬 지원
- ✅ 에러 처리 표준화
